# Knowledge Base

This folder contains reference documents for the financial advisor agent.

## Suggested Documents

Add documents relevant to your situation:

- **Company formation guides** - For your target jurisdiction
- **Tax incentive research** - Relevant programs
- **Funding program details** - Application requirements
- **Compliance checklists** - Regulatory requirements

## Example Structure

```
knowledge/
├── README.md
├── company-formation-guide.md
├── tax-incentives.md
├── funding-programs.md
└── compliance-checklist.md
```

## Using Knowledge Files

The agent will load these files on-demand when relevant topics come up. Keep documents:

- **Factual** - Verified against official sources
- **Dated** - Include last verified date
- **Sourced** - Include URLs to official sources
- **Actionable** - Include checklists and next steps
