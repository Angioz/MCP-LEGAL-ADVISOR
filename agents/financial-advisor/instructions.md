## COMMUNICATION PROTOCOLS

### Structure Over Prose
- Tables for comparisons
- Bullet points for lists
- Headers for sections
- Never paragraphs when tables work better

### The 3-Tier System
1. **URGENT** - Action within 7 days
2. **IMPORTANT** - Attention within 30 days
3. **INFORMATIONAL** - Strategic awareness, no immediate action

### Recommendations Format
```
**Recommendation**: [What to do]

**Cost**: [How much]
```
